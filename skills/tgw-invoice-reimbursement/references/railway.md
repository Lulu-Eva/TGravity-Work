# Railway Tickets

## 识别特征

铁路电子客票通常包含：

```text
电子发票（铁路电子客票）
买票请到12306
中国铁路祝您旅途愉快
票价:￥96.00
购买方名称:深圳璐途文化科技有限公司
```

判断函数：

```text
文本包含 铁路电子客票 / 买票请到12306 / 中国铁路 即按铁路票处理
```

## 字段映射

| 输出字段 | 取值规则 |
|---|---|
| 发票号码 | `发票号码:...` |
| 发票种类 | 固定 `铁路电子客票` |
| 开票日期 | `开票日期:YYYY年MM月DD日` |
| 销售方 | 固定 `中国铁路` |
| 货物、应税劳务及服务 | `铁路运输服务 出发站-到达站 车次`；补票写 `补票 ...`；退票写 `退票费 ...` |
| 价税合计 | 优先 `票价`；退票场景取 `退票费` |

## 项目名称

示例：

```text
铁路运输服务 北京北站-张家口站 G2465
铁路运输服务 北京北站-呼和浩特东站 G2465
铁路运输服务 呼和浩特东站-呼和浩特站 C657
补票 呼和浩特站-托克托东站 C657
退票费 深圳北站-福州南站 D2431
```

不同铁路 PDF 的站点排列不完全一致。每次整理后，至少抽样复查 4 张铁路票的路线和车次。
