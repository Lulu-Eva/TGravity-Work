# Video Indexer Validation

## 验收命令

```bash
python3 scripts/video_pipeline.py validate --workspace "<项目根目录>"
```

## 必须检查

- 视频目录中可识别的视频数量。
- `video_inventory.csv` 是否覆盖全部视频。
- `video_material_draft.csv` 或 Excel 是否覆盖全部视频。
- 必填字段是否为空。
- 建议切片时间码是否符合 `HH:MM:SS-HH:MM:SS` 或包含待复核说明。
- 建议切片时间码是否超出视频时长。
- 输出表格中是否存在不存在的原始视频路径。

## 必填字段

- `素材编号`
- `文件名`
- `路径`
- `时长`
- `素材类型`
- `可用性`
- `音频来源/置信度`
- `主要话题`
- `具体内容`
- `建议切片时间码`
- `画面备注`
- `剪辑建议`

## 失败分级

| 级别 | 含义 | 处理 |
|---|---|---|
| error | 表格无法使用或覆盖不完整 | 修复后重新导出 |
| warning | 可交付但需要人工复核 | 在总结中列出 |
| info | 正常提示 | 可继续 |

## 不能通过验收的情况

- 表格少于视频总数。
- 关键字段大量为空。
- 时间码明显超出素材时长。
- 把低置信 ASR 当作高置信结论。
- 真实素材路径指向 Skill 仓库内部。

```text
- 已对齐逐字稿片段：

- 输出文件：
```
