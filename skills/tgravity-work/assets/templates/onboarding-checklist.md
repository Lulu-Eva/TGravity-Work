# TGravity Work Skill 新员工学习清单

## 第一轮必须会

- [ ] 我知道开展新工作前，要先使用 `/tgravity-workcheck`。
- [ ] 我知道它会先定义工作，再拆分 AI 和人类各自做什么。
- [ ] 我会用口喷或语音转文字，把一堆工作流水交给 `/tgravity-daily` 整理。
- [ ] 我知道 `/tgravity-search` 用于公开互联网资料搜索，需要单独配置 Perplexity API Key。
- [ ] 我知道报价、签约、合同、回款异常、对外承诺必须让人判断。
- [ ] 我知道只有明确说保存，真实资料才写入本地。
- [ ] 我知道 `/tgravity-export` 是导出 TGravity 资产，不是扫描整台电脑。

## 工作分配填空

```text
我现在想做【事情】。
这件事和【对象/项目/平台】有关。
我希望最后得到【文档/表格/话术/名单/决策】。
我不确定哪些该我做，哪些该 AI 做。
```

明天我要做【下一步】。
